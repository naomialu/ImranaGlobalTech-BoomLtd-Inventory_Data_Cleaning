Data Cleaning &amp; Quality Improvement – Inventory Dataset

Data Issues:

Duplicates
Incorrect formats
Data type errors

Cleaning Process:
Duplicate removal
Format standardization
Data validation
Error correction

Tools Used:
Microsoft Excel

Business Outcome:
Clean, reliable dataset ready for reporting and analysis.
